# 暹罗商城-java b2c商城系统

#### Description
暹罗商城是一款Java b2c商城系统，该系统包含用户端、总管理后台，用户下单、快递配送等核心功能，同时提供会员中心和邀请好友注册返现等增值服务，以拓展客源。基于 Spring Boot + MyBatis Plus + Vue & Element + uni-app 技术栈进行开发；
